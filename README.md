# AliControl - Fintech

O **AliControl** é uma solução de Fintech focada na democratização do controle financeiro pessoal. O sistema foi concebido para transformar a maneira como as pessoas gerenciam suas finanças diárias, centralizando gastos, receitas e planejamentos em uma experiência visualmente intuitiva, clara e livre de distrações burocráticas.

## Funcionalidades e Diferenciais do Projeto
*   **100% Livre de JavaScript**: Arquitetura construída estritamente com HTML e CSS puros, eliminando scripts pesados e garantindo um carregamento instantâneo.
*   **Totalmente Responsivo**: Interface e experiência otimizadas de forma fluida para dispositivos móveis (smartphones e tablets) e desktops, utilizando conceitos modernos de mídia e layouts flexíveis.
*   **Identidade Visual Única**: Cores, tipografia, espaçamentos e componentes padronizados sob uma paleta de cores moderna e profissional (baseada em tons de Indigo e Emerald).
*   **Portabilidade Avançada**: Código estático e limpo, pronto para ser executado instantaneamente de forma local em qualquer navegador ou sistema operacional.

## Tecnologias Utilizadas
*   HTML5 (Estruturação semântica e acessibilidade)
*   CSS3 (Variáveis nativas, Flexbox, CSS Grid e Media Queries com base conceitual na arquitetura de classes do framework Tailwind CSS)
